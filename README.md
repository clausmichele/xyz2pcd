# xyz2pcd
Convert Point Clouds from .xyz to .pcd file format.
Tested on Ubuntu 18.04 with Point Cloud Library (PCL) 1.9.

# Requirements:
```
pcl (tested with 1.9)
boost
```

# Build instructions:
```
mkdir build
cd build
cmake ..
make
```

# Instructions:
Run the compiled program:
```
./xyz2pcd
```
And insert the path of your .xyz file.
